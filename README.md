## How to start

- Clone the repo  
> git clone https://github.com/szymondlugolecki/unstuck.io.git
- Install the dependencies
> cd unstuck.io  
> npm install
- Update the Firestore credentials in src/App.tsx
- Use
```
> npm start - development
> npm run build - build
```
